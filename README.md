# maildir-addresses

https://github.com/rrthomas/maildir-addresses  
© Reuben Thomas <rrt@sc3d.org>  

maildir-addresses is Scan a maildir tree for email addresses in From/To/Cc headers.


## Installation

Install with `pip`: `pip install maildir_addresses`.
